Commands

help

Shows brief commands description.

invite

Sends bot invite link to direct messages.

search

Returns formatted embed, for artist/album/track info request. Example: <<search gorillaz, <<search the white album.

listen

Starts command, which lasts for a specified period of time and checks current user's Spotify discord activity. Returns brief music taste analysis in embed.

Listen command has 2 different call methods

  • <<listen # number
Monitors your spotify activity for (# number) minutes. Example: <<listen 5 (command invokes for 5 minutes).
  • <<listen # number @user
Monitors spotify activity of @user for (# number) minutes. Example: <<listen 5 @GamerBoy51 (command invokes for 5 minutes and monitors GamerBoy51's activity).

IMPORTANT NOTE: Currently, no Game/Spotify/Twitch activity can be displayed on IOS and Android Discord versions. So if you use Discord on your IOS/Android device, sadly, there is no way "listen" command will work for you.
